Enable Internet Access

Right now your server is only reachable on your local WiFi. Friends on different networks (mobile data, different WiFi) need a public URL. Set up a free ngrok tunnel in 3 steps:

1
Install ngrok package
In your project folder, run:
npm install @ngrok/ngrok copy
2
Get a free ngrok token
Sign up free at dashboard.ngrok.com → Your Authtoken → copy the token
3
Start server with token
Replace YOUR_TOKEN and restart:
NGROK_AUTHTOKEN=YOUR_TOKEN node server.js copy
On Windows (PowerShell):
$env:NGROK_AUTHTOKEN="YOUR_TOKEN"; node server.js copy
💡 After restart, come back to this page — the public URL will appear automatically and you can share it with anyone.
Password Guide
🔑
Driver Password
Your personal login password. Only you know this. Used to log in to the driver panel.
🧑‍✈️ Drivers only
🚌
Route Password
Shared password for the whole route. Share this with commuters AND other drivers joining the same route.
👥 Everyone on route

⚠️ A second driver joining the same route needs: a unique username, their own personal password, and the same route ID + route password as the first driver.
